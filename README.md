### **Vision-Language-Action (VLA) Robotic Navigation**
*Autonomous perception-to-action pipeline using Multimodal Transformers.*

- **Core Innovation:** Fusing YOLOv8 (Perception) + Phi-3 (Reasoning) + ROS2 Nav2 (Action).
- **Architecture:** Trained a Multimodal Fusion Transformer to interpret natural language ("Go to the blue cup") and output velocity commands.
- **Hardware/Sim:** Deployed on TurtleBot3 via Gazebo with NVIDIA CUDA acceleration.
