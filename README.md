## 🚀 Featured Projects

### 🤖 AI‑Powered Vision‑Language‑Action Robotics System  
*[GitHub](https://github.com/Sunkeerth/Vision-Language-Action-Based-Robotic-Navigation-System-with-Multimodal-Transformer-Reasoning)*

An end‑to‑end VLA pipeline that enables a robot to perceive its environment, understand natural language, and execute complex tasks.

- **Perception**: YOLOv8 object detection + CLIP for relational understanding (e.g., “cup on table”) → structured scene graphs.  
- **Memory**: Persistent world state stored in a memory module for long‑term reasoning.  
- **Planning**: LangChain orchestration layer combines perception, memory, and an LLM (Phi‑3) for high‑level task decomposition.  
- **Execution**: Low‑level actions sent to ROS2 Nav2 for navigation in Gazebo with TurtleBot3.  
- **Robustness**: Failure handling and dynamic re‑planning for real‑world scenarios.

**Tech Stack:** `ROS2` `YOLOv8` `CLIP` `LangChain` `Phi-3` `Gazebo` `Nav2` `Python`

### 🚁 Autonomous Drone Communication & Agriculture System  
*[Drive Demo](https://www.linkedin.com/posts/sunkeerth-ab14b3337_nidar-dronetechnology-bitm-ugcPost-7424705466426941440-QX82?utm_source=share&utm_medium=member_desktop&rcm=ACoAAFSc9WUB3ulIpeEaNNKqDPDRei5i89hIh98)*

A robust communication system for scanning and spraying drones using ESP32/ESP8266 with ESP‑NOW and UART for real‑time control in agricultural fields.

- **Hardware**: ESP32‑based telemetry radios, GPS, failsafe logic.  
- **Protocol**: Lightweight custom packets with ACKs, retries, and sequence numbers.  
- **Field Performance**: Reliable over 90×90 m with Wi‑Fi (2.4 GHz) and external antennas.

**Tech Stack:** `ESP32` `ESP-NOW` `UART` `GPS` `Arduino` `Python (simulation)`
